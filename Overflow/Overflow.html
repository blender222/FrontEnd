前端三大框簡介

由 Google 前員工 Evan You 於2014年開發，是一個開源、輕量級、以視圖(View)層為基礎發展的 JavaScript library，設計理念是易用、靈活、高性能，入門門檻較低，所需要的背景知識與概念不需要像 Angualr 或 React 只要有基本的 HTML、CSS 以及 JavaScript 的知識，就可以快速上手。

由 Facebook 開發，React 捨棄傳統的 HTML 開發模式，改由 JavaScript 來全面代管 HTML 的渲染與 DOM 操作。開發者不必先寫好 HTML 基礎結構後再用 JavaScript 操作 HTML DOM，而是將該如何組合與渲染 UI 以及 UI 的互動事件，都聲明在 JavaScript 中，然後 React 會根據你的定義將其動態的產生並操作真正的 HTML DOM，以實現 UI 畫面的渲染與變更。

由 Google 開發，最初發表於2010，是一個發展已經很成熟、基於 TypeScript 的開源框架，在貢獻者方面有強大後盾，Angular 相較於其他框架，是一個大而全、完整的平台，從開發到測試都一手包辦，但學習曲線較陡峭，要掌握 Angular 需要了解 TypeScript 和 MVC 的相關概念。
